Introduo

O projeto realizado tem como objetivo prever os preos de aes de vrias empresas do S&P500.

Para isso utilizamos algoritmos de aprendizagem clssicos (AdaBoost, XGBoost e LGBM) e de deep learning (LSTM).

Pretende-se, tambm, selecionar os melhores stocks para o investimento dirio, de forma a maximizar os retornos financeiros, razo pela qual recorremos ao Monte Carlo como mtodo de otimizao.

Etapas realizadas

-> Extrao dos dados relativos a todas as empresas presentes no S&P 500;

-> Pr-processamento dos dados;

-> Realizao de Feature Engineering;

-> Realizao de uma anlise explorativa dos dados;

-> Seleo e implementao dos algoritmos a utilizar: AdaBoost, XGBoost, LGBM, LSTM.

-> Implementao do Monte Carlo como mtodo de otimizao;

-> Avaliao do desempenho dos modelos;

-> Anlise dos resultados.

def get_sp500_tickers():
    #URL da pgina da Wikipdia para a lista de empresas S&P 500
    url = "https://en.wikipedia.org/wiki/List_of_S%26P_500_companies"

    #Utilizar o pandas para ler a tabela que contm os tickers
    tables = pd.read_html(url)
    df = tables[0]

    #Extrair a coluna de smbolos de ticker
    tickers = df['Symbol'].tolist()

    #Remove as linhas com tickers em falta
    tickers = [ticker for ticker in tickers if ticker != '']

    return tickers

def download_stock_data(symbols, start_date, end_date, data_dir):
    for symbol in symbols:
        try:
            logger.info(f"Downloading data for {symbol}...")
            data = yf.download(symbol, start=start_date, end=end_date)

            #Guardar os dados num ficheiro CSV
            file_path = data_dir / f"{symbol}.csv"
            data.to_csv(file_path)
            logger.info(f"Data for {symbol} saved to {file_path}")
        except Exception as e:
            logger.error(f"Failed to download data for {symbol}: {e}")

input_folder = r"C:\Users\beatr\Music\updated_data"
output_folder = r"C:\Users\beatr\Music\new_updated_data"

#Iterar sobre todos os ficheiros na pasta de entrada
for filename in os.listdir(input_folder):
    if filename.endswith(".csv"):
        file_path = os.path.join(input_folder, filename)

        df = pd.read_csv(file_path)

        #Adicionar a coluna "Ticker" com o nome do ficheiro
        ticker = os.path.splitext(filename)[0]
        df["Ticker"] = ticker

        #Renomear e reorganizar as colunas
        df.columns = ["Date", "Adj Close", "Close", "High", "Low", "Open", "Volume", "Ticker"]
        df = df[["Date", "Adj Close", "Close", "High", "Low", "Open", "Volume", "Ticker"]]

        #Remover as duas primeiras linhas
        df = df.iloc[2:]

        #Guardar o ficheiro processado na pasta de sada
        output_path = os.path.join(output_folder, filename)
        df.to_csv(output_path, index=False)

        print(f"Processado: {filename} -> {output_path}")

Indicadores Tcnicos (Feature Engineering)

Em relao ao indicadores tcnicos, optmos por calcular os indicados seguidamente:

  • Mdias Mveis Simples (SMA_20, SMA_50)*: Calculam a mdia do preo ajustado dos ltimos 20 e 50 perodos, respetivamente. So usados para identificar tendncias e suavizar flutuaes de curto prazo.

  • Mdia Mvel Exponencial (EMA_20): D mais peso aos preos recentes em comparao SMA, sendo til para identificar tendncias com maior sensibilidade a alteraes recentes.

  • RSI (Relative Strength Index): Mede a fora relativa de um ativo ao longo de 14 perodos, ajudando a identificar condies de sobrecompra ou sobrevenda.

  • Bandas de Bollinger: Usam uma mdia mvel de 20 perodos (BB_Middle) e dois desvios padro para calcular os limites superior e inferior (BB_Upper, BB_Lower), que ajudam a avaliar a volatilidade e a identificar possveis reverses.

  • Momentum (Momentum_10): Mede a variao do preo ajustado em 10 perodos, avaliando a fora e a direo do movimento dos preos.

  • ATR (Average True Range): Calcula a mdia da amplitude diria de preos ao longo de 14 perodos, representando a volatilidade mdia.

  • Volatilidade (Desvio Padro de 20 perodos): Mede a disperso do preo ajustado em 20 perodos, indicando o grau de incerteza ou risco.

  • Retornos dirios (Daily_Return): Calculam a variao percentual diria do preo ajustado, usados para a anlise de desempenho e da volatilidade.

  • Retornos futuros (Future_Return): Antecipam a variao percentual futura do preo ajustado, servindo como varivel alvo para modelos de previso.

  • Volatilidade Mvel (Rolling_Volatility): Mede o desvio padro dos retornos dirios em janelas mveis de 10 dias, fornecendo uma viso mais detalhada da volatilidade ao longo do tempo.

Referncias:

Technical Analysis of the Financial Markets" por John Murphy (1999)

df['SMA_20'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: x.rolling(window=20).mean())
df['SMA_50'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: x.rolling(window=50).mean())
df['EMA_20'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: x.ewm(span=20, adjust=False).mean())

# Preencher valores NaN gerados pelas mdias mveis
df['SMA_20'].fillna(method='ffill', inplace=True)
df['SMA_50'].fillna(method='ffill', inplace=True)
df['EMA_20'].fillna(method='ffill', inplace=True)

# 2. RSI (Relative Strength Index) - Padro: 14 perodos
df['RSI'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: ta.momentum.rsi(x, window=14))

# 3. Bandas de Bollinger (20 perodos, 2 desvios padro)
df['BB_Middle'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: x.rolling(window=20).mean())
df['BB_Upper'] = df['BB_Middle'] + 2 * df.groupby('Ticker')['Adj Close'].transform(lambda x: x.rolling(window=20).std())
df['BB_Lower'] = df['BB_Middle'] - 2 * df.groupby('Ticker')['Adj Close'].transform(lambda x: x.rolling(window=20).std())

# Preencher valores NaN das Bandas de Bollinger
df['BB_Middle'].fillna(method='ffill', inplace=True)
df['BB_Upper'].fillna(method='ffill', inplace=True)
df['BB_Lower'].fillna(method='ffill', inplace=True)

# 4. Momentum (10 perodos)
df['Momentum_10'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: x.diff(periods=10))

# 5. ATR (Average True Range, 14 perodos)
df['ATR'] = ta.atr(df['High'], df['Low'], df['Close'], length=14)

# 6. Volatilidade (Desvio Padro de 20 perodos)
df['Volatility'] = df.groupby('Ticker')['Adj Close'].transform(lambda x: x.rolling(window=20).std())

# Preencher valores NaN da volatilidade
df['Volatility'].fillna(method='ffill', inplace=True)

# Calculate daily returns
df['Daily_Return'] = df.groupby('Ticker')['Adj Close'].pct_change()

# Calculate future returns as the target
df['Future_Return'] = df.groupby('Ticker')['Adj Close'].pct_change().shift(-1)

# Rolling volatility (e.g., 10-day standard deviation)
df['Rolling_Volatility'] = df.groupby('Ticker')['Daily_Return'].rolling(window=10).std().reset_index(0, drop=True)
